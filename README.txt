Working on Ayehu GitHub Community exercise.

The following is a Python 3.9.6 verified Activity which accepts input "Python" and returns "Hello to Python's World!!!" via invocation of the Python interpreter. The user may modify the default value so that the result produced reads as "Hello to <user defined> World!!!". The activity is called CheckPythonSetup and the category is HelloWorld. This submission is in response to Ayehu Academy Dev Ayehu GitHub Community verification of a submission.

Help Section:
This is a custom Actions Express Activity providing a Hello World check to confirm the Python interpreter can be executed to do work in an activity. The code was tested with Python39 (3.9.6) installed as Administrator for all users to access in C:\Program Files using the option for the installer to update the PATH variable on Windows 10.
