<br>#     Zoom</br>
<br>List Tracking Fields</br>
<br>[Tracking fields](https://support.zoom.us/hc/en-us/articles/115000293426-Scheduling-Tracking-Fields) allow you to analyze usage by various fields within an organization. Use this API to list all the tracking fields on your Zoom account.
**Scopes:** `trackingfield:read:admin`
 
**Prerequisites:**
* Business, Education, API or higher plan</br>
<br>Method: Get</br>
<br>OperationID: trackingfieldList</br>
<br>EndPoint:</br>
<br>/tracking_fields</br>
