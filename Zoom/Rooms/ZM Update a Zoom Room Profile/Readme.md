<br>#     Zoom</br>
<br>Update a Zoom Room Profile</br>
<br>Update basic information on a specific Zoom Room in a Zoom account.

**Scopes**: `room:write:admin` </br>
<br>Method: Patch</br>
<br>OperationID: updateRoomProfile</br>
<br>EndPoint:</br>
<br>/rooms/{roomId}</br>
