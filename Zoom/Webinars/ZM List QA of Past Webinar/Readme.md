#     Zoom


List Q&A of Past Webinar

The [Question  Answer (QA)](https://support.zoom.us/hc/en-us/articles/203686015-Getting-Started-with-Question-Answer) feature for Webinars allows attendees to ask questions during the Webinar and for the panelists, co-hosts and host to answer their questions.
Use this API to list QA of a specific Webinar.

**Prerequisites:**
* [Webinar license](https://zoom.us/webinar)
**Scopes**: `webinar:read:admin`, `webinar:read`
 





Method: Get

OperationID: listPastWebinarQA

EndPoint:

/past_webinars/{webinarId}/qa
