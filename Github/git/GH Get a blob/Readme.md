<br>#     Github</br>
<br>Get a blob</br>
<br>The `content` in the response will always be Base64 encoded.

_Note_: This API supports blobs up to 100 megabytes in size.</br>
<br>Method: Get</br>
<br>OperationID: git/get-blob</br>
<br>EndPoint:</br>
<br>/repos/{owner}/{repo}/git/blobs/{file_sha}</br>
